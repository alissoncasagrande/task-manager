# Contributing to Task Manager

At the moment, this project is being developed by **Alisson Guindo Casagrande** only.  
Contributions are not open yet, but in the future I plan to allow external collaborators.

## Guidelines (for future contributions)

When contributions are open:
- Fork the repository.
- Create a new branch for your feature or fix.
- Write clear commit messages.
- Submit a Pull Request with a detailed description of the changes.

## Code Style

- Follow Python best practices (PEP8).
- Keep functions small and focused.
- Document code with comments when necessary.

## Issues

If you find a bug or have a suggestion, please open an issue in the repository.
